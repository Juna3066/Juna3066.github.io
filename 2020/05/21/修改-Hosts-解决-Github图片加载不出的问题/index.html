为什么修改了host文件就可以解决标题描述的问题呢

人输入网址（协议+域名+端口号+路径）,然后DNS服务器把网址中的域名映射成对应的ip地址，然后就可以访问对应ip的主机资源了。
在这个过程中，DNS解析这一步因为各种因素可能会存在问题，导致映射错误，不能正确访问对应主机资源。
访问github慢，图片不显示等，大概就是这个原因；

通过修改hosts文件,相当于在自己的主机上已经做了映射了，只要你在hosts中追加的github相关的映射正确，那么（DNS解析这一步因为各种因素可能会存在问题）就可以避免，你就可以正常访问github了。
